1. Lancer le programme

Double-cliquez sur le fichier yt_downloader.py pour démarrer.

Une petite fenêtre peut s'ouvrir et se fermer rapidement la première fois. C'est normal : le script vérifie qu'il a tous ses "outils" et les installe si besoin.

Ensuite, la fenêtre principale du programme s'ouvre.

2. Remplir les champs

Vous avez 3 choses principales à faire :

URL (Vidéo, Playlist ou Chaîne) :
Allez sur YouTube, copiez le lien de la vidéo (ou de la playlist) que vous voulez, et collez-le dans ce premier champ.

Dossier de destination :
Cliquez sur le bouton bleu "Parcourir". Une fenêtre s'ouvrira pour vous permettre de choisir où sur votre ordinateur vous voulez enregistrer le fichier (par exemple : sur votre "Bureau", dans "Mes Vidéos" ou "Téléchargements").

Régler les options :

Audio seulement (MP3) : Cochez cette case si vous ne voulez que le son (la musique) et pas la vidéo.

Télécharger la vidéo seule... : Si votre lien est une vidéo qui fait partie d'une playlist, cochez cette case pour ne télécharger que cette vidéo, et non toute la playlist.

Qualité Vidéo : "Source (Best)" est le meilleur choix, il prendra la meilleure qualité disponible (4K, 1080p...).

Codec Vidéo : C'est le réglage le plus important pour la vitesse.

Copier la vidéo (Rapide) : (Choix recommandé). C'est le plus rapide. Le programme prend la vidéo et le son et les colle ensemble. Cela prend quelques secondes.

h264 (Lent) / h265 (Très lent) : Ces options forcent l'ordinateur à re-compresser toute la vidéo. C'est très, très lent (peut prendre 10 minutes pour une vidéo de 3 minutes). Ne les utilisez pas, sauf si le mode "Rapide" ne fonctionne pas.

3. Lancer le téléchargement

Quand tout est prêt, cliquez sur le gros bouton vert "Télécharger".

Étape 1 : Téléchargement
Le bouton va devenir gris et la barre de progression va se remplir. Le script est en train de télécharger les fichiers depuis YouTube.

Étape 2 : Encodage / Fusion
Une fois le téléchargement fini, le texte peut changer en "Encodage / Fusion en cours...". Le programme est en train d'assembler le fichier final.

Si vous avez choisi "Copier (Rapide)", cette étape dure 5 secondes.

Si vous avez choisi "Lent", c'est ici que vous devrez être patient.

Étape 3 : Terminé !
Quand tout est fini, une petite fenêtre "Succès !" apparaîtra. Votre fichier est prêt dans le dossier que vous aviez choisi !

Le bouton redevient vert, prêt pour un autre téléchargement.
