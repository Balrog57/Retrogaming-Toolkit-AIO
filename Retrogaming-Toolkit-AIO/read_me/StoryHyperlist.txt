Enrichissez vos bases de données Hyperspin ou HyperList sans éditer une seule ligne de code manuellment. Ce module fusionne automatiquement une collection de fichiers texte (synopsis, histoires) directement dans votre fichier XML principal.
Il scanne votre dossier de descriptions, associe chaque fichier .txt au jeu correspondant dans la liste, et injecte le contenu dans la balise <story>. Robuste, il gère les différents encodages de texte pour éviter les caractères corrompus et génère un nouveau fichier XML mis à jour, prêt à être intégré dans votre frontend.
