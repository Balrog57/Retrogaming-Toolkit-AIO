Ce module permet de copier des fichiers d'une collection source vers une destination en se basant sur un fichier témoin (modèle) pour identifier les fichiers à copier, tout en préservant la structure relative des dossiers.

Fonctionnement :
1.  **Fichier Témoin** : Sélectionnez un fichier exemple dans votre arborescence source (ex: `C:\RetroFE\collections\MAME\system_artwork\device.png`). Le module utilisera le nom de ce fichier (`device.png`) comme modèle de recherche.
2.  **Dossier Source** : Indiquez la racine de votre collection (ex: `C:\RetroFE`). Le module cherchera toutes les occurrences du fichier modèle à partir de ce dossier.
3.  **Dossier Destination** : Choisissez où copier les fichiers (ex: `D:\Export`). Le module recréera l'arborescence nécessaire.

Exemple :
Si le fichier témoin est `device.png` et qu'il est trouvé dans `C:\RetroFE\collections\SNES\system_artwork\device.png`, il sera copié vers `D:\Export\collections\SNES\system_artwork\device.png`.

Options :
- **Même profondeur uniquement (Mode Strict)** :
    - Activé par défaut.
    - Vérifie que le fichier trouvé se trouve :
        1. À la même "profondeur" (nombre de sous-dossiers) que le témoin.
        2. Dans la même structure de dossier racine.
    - *Exemple* : Si votre témoin est dans `collections\...\device.png`, le script ignorera `layouts\...\device.png` ou `backup\...\device.png`.

Utilisation :
1.  Remplissez les champs "Fichier Témoin", "Source" et "Destination".
2.  Laissez "Même profondeur uniquement" coché pour éviter les faux positifs.
3.  Cliquez sur "Démarrer la Copie".
4.  Suivez la progression dans le journal.
