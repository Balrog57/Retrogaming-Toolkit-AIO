## 2025-05-22 - Keyboard Accessibility in Desktop Tools
**Learning:** Desktop users expect standard keyboard shortcuts (Ctrl+F, Esc, Arrows) for navigation and search. Even in mouse-centric launchers, the absence of 'Escape to clear' or 'Ctrl+F to search' is noticeable.
**Action:** Standardize adding bindings for <Control-f>, <Escape>, and Arrow keys in all list-based GUI views.
